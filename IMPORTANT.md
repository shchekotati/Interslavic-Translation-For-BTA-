# About the Translation 

This translation of the **Better Than Adventure!** mod into the Interslavic language was built upon the existing [Russian localization pack by **skid_ink**](https://modrinth.com/resourcepack/bta!-russian-language-pack). 

Consequently, the text may contain not only newly introduced typos but also phrasing issues or inconsistencies inherited from the original resource pack.

### Key Features:
* **No AI Used:** The Cyrillic version of the translation was crafted entirely by hand.
* **Automated Transliteration:** The Latin version was generated using the [Interslavic Transliterator](https://interslavic.fun). Please note that the Latin text may contain technical or contextual errors caused by the automated conversion.

## Contributing

Contributions are highly welcome! If you spot any grammatical, logical, or contextual errors in either the Cyrillic or Latin text, your help in fixing them would be greatly appreciated.

You can contribute by:
1. Opening an **Issue** to report bugs or typos.
2. Submitting a **Pull Request** (PR) with your fixes. Any commits aimed at improving the quality of the localization are more than welcome!
